# MoveQuest 星轨

MoveQuest 星轨是一个面向大学生的游戏化运动激励 App。产品按移动端体验设计，当前网页链接承载可直接体验的高保真 App：用户选择“走多少步、点亮几个光点”的今日目标，通过现实走动获得“同步次数”，再在星轨小游戏里踩准时机点亮光点。

核心流程：

1. 输入昵称并选择今日目标。
2. 点击“开始走动”，用真实设备运动或演示采集获得同步次数。
3. 角色靠近光点时点击“同步”，完成实时判定小游戏。
4. 好友助攻可补充同步次数，降低中断后的放弃概率。
5. 通过点亮进度、连续完成和小成就形成持续反馈。

本地运行：

```bash
node server.mjs
```

然后打开 `http://127.0.0.1:4173/`。
