# MoveQuest 步步兽

MoveQuest 步步兽是一个面向大学生的游戏化运动激励 App。产品按移动端体验设计，核心机制是“开屏介绍 -> 选择宠物 -> 步数阶段目标 -> 解锁食物 -> 投喂互动 -> 校园探索 -> 好友协作 -> 长线成就与进化”。

核心流程：

1. 开屏页解释“走路变食物、投喂促成长、连续运动促进化”。
2. 用户从多只宠物中选择伙伴，设置头像、昵称、宠物名和今日运动目标。
3. 进入 App 后顶部常驻用户头像、用户名和等级，通过底部导航切换小屋、投喂、探索、好友、成长五个功能区。
4. 开始走动，步数达到阶段节点后解锁食物或互动道具。
5. 回到投喂页喂食，宠物会通过 3D 动画、表情、亲密度和成长值反馈。
6. 今日喂饱后不会结束体验，继续走路可获得旅行便当，派宠物去图书馆、操场、食堂等校园地点带回明信片。
7. 好友页提供串门礼物、约走提醒、双人投喂挑战和宠物串门，但不能替代自己的步数目标。
8. 成长页展示多日进化门槛、成长进度和成就墙，避免一天内完成全部进化。

本地运行：

```bash
node server.mjs
```

然后打开 `http://127.0.0.1:4173/`。
