# MoveQuest 步步兽

MoveQuest 步步兽是一个面向大学生的游戏化运动激励 App。产品按移动端体验设计，核心机制是“步数阶段目标 -> 解锁食物 -> 投喂虚拟宠物 -> 宠物成长反馈”。

核心流程：

1. 输入昵称、宠物名字，并选择今日运动目标。
2. 开始走动，步数达到阶段节点后解锁食物或互动道具。
3. 回到宠物小屋投喂，宠物会通过表情、动画、亲密度和成长值反馈。
4. 好友宠物可串门送小零食，但不能替代自己的步数目标。
5. 通过连续天数、今日喂饱和阶段成就提升持续运动频率。

本地运行：

```bash
node server.mjs
```

然后打开 `http://127.0.0.1:4173/`。
