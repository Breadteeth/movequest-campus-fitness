# MoveQuest 步步兽

MoveQuest 步步兽是一个面向大学生的游戏化运动激励 App。产品按移动端体验设计，核心机制是“开屏介绍 -> 选择宠物 -> 步数阶段目标 -> 解锁食物 -> 投喂互动 -> 宠物进化”。

核心流程：

1. 开屏页解释“走路变食物、投喂促成长、连续运动促进化”。
2. 用户从多只宠物中选择伙伴，输入昵称和宠物名，并选择今日运动目标。
3. 进入 App 后通过底部导航切换小屋、投喂、好友、成长四个功能区。
4. 开始走动，步数达到阶段节点后解锁食物或互动道具。
5. 回到投喂页喂食，宠物会通过 3D 动画、表情、亲密度和成长值反馈。
6. 好友宠物可串门送小零食，但不能替代自己的步数目标。
7. 成长页展示进化阶段、成长进度和今日成就。

本地运行：

```bash
node server.mjs
```

然后打开 `http://127.0.0.1:4173/`。
