# MoveQuest 步步兽

面向大学生的游戏化运动激励 App。产品形态按移动端 App 设计，通过“真实步数 -> 解锁食物 -> 小屋投喂 -> 校园探索 -> 明信片与成就 -> 长期进化”的闭环，提升运动频率。

当前版本重点：

1. 开屏介绍、账号注册、密码、昵称、宠物选择和自定义头像弹层。
2. 小屋作为主场景，投喂、探索、相册都封装为入口卡片，点击后进入底部详情面板。
3. 好友页只展示轻量入口，约走、双人便当、串门和好友动态在详情面板中展开。
4. 成长页移除重复个人档案，只保留进化摘要、阶段成就和详情入口。
5. 明信片可点击打开，展示宠物带回来的来信内容。
6. 全局宠物浮窗常驻三大栏目，可收起和恢复。

本地运行：

```bash
node server.mjs
```
