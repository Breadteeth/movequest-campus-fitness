# MoveQuest 步步兽

MoveQuest 步步兽是一个面向大学生的游戏化运动激励 App。产品按移动端体验设计，核心机制是“注册账号 -> 上传裁切头像 -> 选择宠物 -> 今日运动主线 -> 步数阶段目标 -> 解锁食物 -> 投喂互动 -> 校园探索 -> 好友协作 -> 长线成就与进化”。v6 版本重做了视觉系统，减少并列卡片堆叠，以小屋首页的运动主线作为核心操作入口。

核心流程：

1. 开屏页解释“走路变食物、投喂促成长、连续运动促进化”。
2. 用户创建校园账号和密码，可上传头像并通过缩放/位置控制完成圆形裁切。
3. 用户从多只宠物中选择伙伴，设置昵称、宠物名和今日运动目标。
4. 进入 App 后顶部常驻用户头像、用户名、账号和等级，通过底部导航切换小屋、投喂、探索、好友、成长五个功能区。
5. 小屋页用“走路 -> 投喂 -> 探索”的今日主线引导新用户，帮助页解释产品玩法。
6. 开始走动，步数达到阶段节点后解锁食物或互动道具。
7. 回到投喂页喂食，宠物会通过 3D 待机移动、表情、亲密度和成长值反馈。
8. 今日喂饱后不会结束体验，继续走路可获得旅行便当，派宠物去图书馆、操场、食堂等校园地点带回明信片。
9. 好友页提供串门礼物、约走提醒、双人投喂挑战和宠物串门，但不能替代自己的步数目标。
10. 成长页展示多日进化门槛、成长进度和成就墙，避免一天内完成全部进化。

本地运行：

```bash
node server.mjs
```

然后打开 `http://127.0.0.1:4173/`。
