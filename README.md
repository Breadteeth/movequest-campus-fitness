# MoveQuest

MoveQuest 是一个面向大学生的游戏化运动激励 App Demo。正式产品形态按移动 App 设计，网页链接用于测评时直接打开和演示。

产品主流程：

1. 新用户选择运动目标，进入今日任务。
2. 用户完成低门槛任务，获得 XP、金币和连续打卡反馈。
3. 跑图页用模拟 GPS 展示校园运动轨迹、虚拟补给点和奖励结算。
4. 战队页通过宿舍/社团熟人圈推动互相提醒、组队挑战和排行榜。
5. 成就页承接 Battle Pass、徽章、金币兑换和商业化入口。

本地运行：

```bash
node server.mjs
```

然后打开 `http://127.0.0.1:4173/`。
